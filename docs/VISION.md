# VMS Vision

## Mission

To build a trusted digital ecosystem for mapping, surveying, land valuation, and property information in Nepal.

## Long-term Goal

Create a platform where everyone can:

- Explore maps
- View pinned locations
- Access public property records
- View government valuation information
- Access government contacts
- Use professional valuation tools
- Contribute verified information

## Principles

- Public First
- Transparency
- Accuracy
- Simplicity
- Security
- Scalability

## Motto

**Mapping Nepal Together**
